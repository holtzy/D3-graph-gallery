An introduction to d3.js in 10 basic examples.




D3.js is a JavaScript library for manipulating documents based on data. It allows to build absolutely any type of visualization, with no limits. This documents is a quick intro to d3. It displays 10 examples illustrating the key concepts of d3, leading to a first basic scatterplot.

What is HTML?

HTML stands for Hypertext Markup Language. Basically, it is the language behind any website. Web browsers like Mozilla or Safari read this kind of file and translate it in a webpage.

In a HTML file, elements composing the webpage are created, delineated by tags. For instance a title of level 1 is represented by the h1 tag, a paragraph with the p tag, an image by the img tag and so on.

It is impossible to create a d3.js visualization without basic knowledge on html. This tutorial by W3School gives a good overview. Here is a very basic html document: (code on the right, result on the left)

← Edit me! 
<!DOCTYPE html>

<!-- Add a title -->
<h1>First html document</h1>

<!-- Add a bit of text -->
<p>This is my first sentence</p>

<!-- Add a link -->
<p>This is <a href="https://www.d3-graph-gallery.com">a link to the d3 graph gallery</a></p>

Try: Copy and paste the code below in a local file. Open it with a browser. You've created your first website!

Custom document style with CSS

CSS stands for Cascading Style Sheet. It allows to apply specific styles to the elements created in html before.

As for html, it is impossible to create a d3.js visualization without basic knowledge on css. W3School provides a tutorial on css that is a requirement IMO.

Let's apply a modification to the previous document using css:

← Edit me! 
<!DOCTYPE html>

<!-- Define style of the elements that have the class `inGreen` -->
<style>
.inGreen { color: green; }
</style>

<!-- Add a title -->
<h1 class="inGreen">First html document</h1>

<!-- Add a bit of text -->
<p>This is my first sentence</p>

<!-- Add a link -->
<p>This is <a href="https://www.d3-graph-gallery.com">a link to the d3 graph gallery</a></p>

Try: Copy and paste the code below in a local file. Open it with a browser. You've created your first website!

Build shapes with SVG

CSS stands for Scalable Vector Graphic. It is an vector image format. Basically, it is a language that allows to build shapes with code.

A d3.js chart is actually a set of svg shapes put together. For instant, a scatterplot is just composed by several circles as the one shown below.

This document of the d3 graph gallery showcases the different shapes offered by Svg.

← Edit me! 
<!DOCTYPE html>
<!-- Add a title -->
<h1 class="inGreen">First html document</h1>

<!-- Add a bit of text -->
<p>This is my first sentence</p>

<!-- Add a svg shape -->
<svg>
  <circle style="fill: #69b3a2" stroke="black" cx=50 cy=50 r=40></circle>
</svg>

Try: Copy and paste the code below in a local file. Open it with a browser. You've created your first website!
