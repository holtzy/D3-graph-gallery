Drawing axis in d3.js




The d3.js allows to draw shapes, that together build a graph. This shapes are most of the time svg elements. It is important to have a few basics concerning svg.

Circle

Allows to draw a circle. Works like that:

// create svg element
var svg = d3.select("#circle")
  .append("svg")
    .attr("width", 1000)

// add a circle
svg
  .append("circle")
  .attr("cx", 300)
  .attr("cy", 70)
  .attr("r", 60)
  // More customization
  .attr("fill", "#69b3a2")
  .attr("stroke", "black")
  .attr("stroke-width", 5)

Note that this is exactly the same as writing this in your html file:

<circle cx="300" cy="70" r="60" fill="#69b3a2" stroke="black" stroke-width="5"></circle>

svg

path

Allows to draw a path. A bit more complicated, but very powerful. Works like that:

// create svg element
var svg = d3.select("#path")
  .append("svg")
  .attr("width", 1000)

  // add a path
  svg
    .append("path").attr("stroke","black")
    .attr("d","M0 0 L 150 150 L300 100")

Note that this is exactly the same as writing this in your html file:

<path stroke="black" d="M0 0 L 150 150 L300 100"></path>
